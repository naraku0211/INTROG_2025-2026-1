Installation
Clone the repository:

bash
git clone https://github.com/naraku0211/INTROG_2025-2026-1.git
cd INTROG_2025-2026-1
Open the solution file (.sln) in Visual Studio or your preferred IDE.

Restore NuGet packages:

dotnet restore
Build the solution:

dotnet build
Run the application:
